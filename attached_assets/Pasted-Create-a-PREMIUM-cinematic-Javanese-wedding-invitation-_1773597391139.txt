FLOWER PETAL SYSTEM

Create a reusable component:

FlowerPetals.jsx

Use Framer Motion to render falling jasmine petals.

Petals should:

fall slowly from top
rotate slightly
fade out near bottom
spawn randomly across screen width

---

FLOWER CONFIG FILE

Create a configuration file:

src/config/flowerConfig.js

Example:

export const flowerConfig = {
enabled: true,
petalCount: 20,
speed: 18,
size: 22,
opacity: 0.7,
rotation: true
}

FlowerPetals component must read settings from this config.

This allows easy animation control later.

FLOWER LAYER
